paper-refiner tool · 上传 docx → 选择段落与提示词 → 生成改写 → 人工审核后导出

1. 上传原稿

2. 选择段落与 Prompt

勾选段落与 Prompt 设置会自动保存；刷新页面或重启服务后，将自动恢复上次会话，可直接点「生成改写」。

仅显示正文流中的段落（表内段落不在列表）。可一次勾选多段（默认最多 1000 段，环境变量 PAPER_REFINER_MAX_BATCH 可调）。点击「生成改写」后，服务端按段落编号从小到大逐段调用 LLM（每段一次请求），完成后在「3. 人工审核」中按序号展示。

3. 人工审核

结果按段落编号升序排列。可编辑右侧「改写稿」；仅「采纳本段」勾选的段落会写入导出 docx。

API 兼容 OpenAI 格式；gptsapi 可使用 OPENAI_API_BASE=https://api.gptsapi.net；DeepSeek 可使用 https://api.deepseek.com。
